## iSystem

> 可视化配置 hosts, 支持 windows , linux 及 osx

### 安装

> `npm install isystem -g`

### 使用
> ` isystem `

当看见 isystem listen at 3005 则说明启动成功

### 帮助  

> 使用 `isystem -h` 来获取帮助           
查看当前版本 ` isystem -V `        
使用2000端口启动 isystem ` (sudo) isystem -p 2000 `         
清除所有config文件 ` isystem -c `
