## iSystem

> 可视化配置 hosts  和 环境变量, 支持 windows , linux 及 osx

### 安装

> `npm install isystem -g`

### 使用
> `windows : isystem`
`osx, linux : sudo isystem`

当看见 isystem listen at 3005 则说明启动成功

### 帮助

> 使用 `isystem -h` 或 `isystem --help`  来获取帮助         
更改端口 ` (sudo) isystem -p 2000 ` 使用2000端口启动 isystem
